# WealthPulse | Personal Financial Dashboard

A premium, high-performance financial tracking dashboard designed for hosting on GitHub Pages.

## Features
- **Total Net Worth Tracking**: Real-time summary of your assets.
- **Portfolio Analytics**: Interactive line charts showing growth over time.
- **Spending Breakdown**: Visual donut charts for monthly expenditure.
- **Market Watchlist**: Keep track of your favorite stocks with live-look indicators.
- **Real Estate Valuation**: Dedicated section for property assets.
- **Premium Design**: Dark mode, glassmorphism, smooth animations, and responsive layout.

## 🚀 Deployment to GitHub Pages

To host this website on your GitHub account for free:

1. **Create a Repository**: Create a new repository on GitHub (e.g., `wealth-pulse`).
2. **Upload Files**: Upload `index.html`, `style.css`, `script.js`, and any assets.
3. **Enable Pages**:
   - Go to **Settings** > **Pages** in your GitHub repository.
   - Under **Build and deployment** > **Branch**, select `main` (or your primary branch) and the `/ (root)` folder.
   - Click **Save**.
4. **View Live**: GitHub will provide a URL (usually `https://username.github.io/wealth-pulse/`) where your dashboard is live.

## Tech Stack
- **HTML5/CSS3**: Modern layouts and glassmorphism.
- **JavaScript**: Dynamic data handling.
- **Chart.js**: High-performance visualizations.
- **Lucide Icons**: Clean, vector-based iconography.
